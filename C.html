C Programming Language Tutorial

The C Language is developed by
Dennis Ritchie for creating system
applications that directly interact with
the hardware devices such as drivers, kernels, etc. C programming is considered as the base for other programming languages, that is why it is known as mother language.
It can be defined by the following ways:
1.Mother language
2.System programming languag
3. Procedure-oriented programming language
4. Structured programming language
5.Mid-level programming language
